# 视频脚本模板库

## 科普类脚本模板

### 结构

```
## [主题] 科普视频

### 基本信息
- 时长：5-8分钟
- 风格：专业但易懂
- 受众：对[主题]感兴趣的大众

### 开场（0:00-0:30）
**[画面]：震撼开场画面**
> [用惊人事实/问题/反差引入]
>
> 例子：「2019年，人类拍下第一张黑洞照片...」

### 主体（0:30-6:00）

#### 第1章：[核心概念1]（1-2分钟）
**[画面]：概念解释动画**
> [旁白：用类比让复杂概念变简单]
>
> 例子：「黑洞不是洞，而是一种极端天体...」

#### 第2章：[核心概念2]（1-2分钟）
**[画面]：可视化演示**
> [旁白：深入但不说教]

#### 第3章：[应用/影响]（1-2分钟）
**[画面]：实例/应用场景**
> [旁白：连接现实]

### 结尾（6:00-7:00）
**[画面]：升华画面**
> [总结金句]
> CTA：「关注我，看更多[主题]内容」
```

## 短视频模板（60秒）

### 黄金3秒法则

```
### 开场（0-3秒）
**[画面]：最有冲击力的画面]
> 「你知道吗？[惊人事实]」

### 核心内容（3-50秒）
**[画面]：快速切换+字幕强调]
> 关键点1：「xxx」
> 关键点2：「xxx」
> 关键点3：「xxx」

### 结尾（50-60秒）
**[画面]：互动引导]
> 「你怎么看？评论区告诉我」
```

## 口播脚本模板

```
## [主题] 口播脚本

### 基本信息
- 形式：真人出镜/无脸口播
- 时长：2-5分钟
- 风格：亲和、专业

### 逐字稿

#### 开场
哈喽，大家好！今天想跟大家聊[话题]...

#### 主体
首先...[第一点]
然后...[第二点]
最后...[第三点]

#### 结尾
以上就是我的分享。如果你觉得有用...
关注我，我们下期再见！
```
