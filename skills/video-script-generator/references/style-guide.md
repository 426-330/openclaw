# 视频语言风格指南

## 口语化技巧

### 原则

- 短句优先
- 避免书面语
- 加入口语词（「其实」「你知道吗」「特别有意思」）

### 改写示例

❌ 书面语：「黑洞是一种引力极强的天体」
✅ 口语化：「说到黑洞，很多人觉得它是个洞，但其实...它是一种引力超级强的天体」

❌ 书面语：「人工智能正在改变我们的生活」
✅ 口语化：「你有没有发现，现在ai已经无处不在了？」

## 节奏控制

### 黄金节奏公式

- 快节奏：信息密度高、剪辑快（展示/种草）
- 中节奏：娓娓道来（科普/故事）
- 慢节奏：沉淀情绪（抒情/总结）

### 断句技巧

- 每句话不超过15字
- 重要信息前停顿
- 数字拆开说（「三百万」→「300万」）

## 情绪把控

### 不同风格的情绪曲线

**科普类**

```
平静 → 小好奇 → 恍然大悟 → 兴趣提升 → 满足感
```

**种草类**

```
共鸣（痛点）→ 期待 → 兴奋 → 紧迫感（限时）→ 行动
```

**故事类**

```
悬念 → 紧张 → 高潮 → 感动/思考 → 留白
```

**教程类**

```
信心建立 → 跟随感 → 成就感 → 鼓励继续
```

## 常用技巧

### 开场钩子模板

- 「你知道吗...[反常识]」
- 「今天教你...[收益前置]」
- 「有个事...[悬念]」
- 「我最近...[真实经历]」

### 结尾金句模板

- 「[核心观点]，这就完了」
- 「记住，[关键点]」
- 「[总结]+关注我」

### 过渡句

- 「说到这...」
- 「但是...」
- 「你知道吗，还有一个更神奇的...」

## 禁用词/表达

❌ 避免：

- 「众所周知」（显得高高在上）
- 「不言而喻」（太书面）
- 「综上所述」（太正式）
- 过于绝对的词（「必须」「一定」）

✅ 建议：

- 用「建议」「可以」「最好」
- 用「很多人会觉得」代替「众所周知」
