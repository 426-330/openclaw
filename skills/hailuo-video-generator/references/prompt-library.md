# 海螺AI提示词库

## 科普类提示词

### 基础知识科普

```
请生成一个[X]分钟的科普视频，主题是[主题]。

要求：
- 语言通俗易懂，适合[目标受众]
- 内容准确严谨，不传播错误信息
- 结构清晰，逻辑连贯

脚本内容：
[粘贴脚本]

画面要求：
- 使用[实拍/动画/数据可视化]风格
- 配合图表和示意图
- 添加文字标注
```

### 热点科普

```
请生成一个[X]分钟的热点科普视频，主题是[热点事件/话题]。

背景：[事件背景介绍]

核心知识点：
1. [知识点1]
2. [知识点2]
3. [知识点3]

画面要求：
- 结合新闻画面和动画解释
- 数据可视化展示
- 专家采访片段（如有）
```
