# 海螺AI视频生成器

在海螺AI官网(https://hailuoai.com/agent)生成视频，并将视频保存到本地。

## 使用场景

当用户需要将视频脚本转换为实际视频时使用，包括：

- 「用海螺AI生成视频」
- 「帮我把脚本做成视频」
- 「打开海螺AI官网生成视频」
- 「下载海螺AI生成的视频」

## 注意事项

### 浏览器服务

OpenClaw 的浏览器工具可能因端口占用而无法自动启动。如果自动打开失败：

1. 手动打开 Chrome/Safari
2. 访问：https://hailuoai.com/agent
3. 手动粘贴脚本并生成视频

### 登录要求

- 海螺AI需要登录账号才能生成视频
- 确保已登录海螺AI账户
- 部分功能可能需要付费积分

## 工作流程

### 第一步：打开海螺AI

访问：https://hailuoai.com/agent

### 第二步：准备脚本

确保视频脚本已生成，格式为：

- 旁白文案
- 画面描述
- 配乐建议

### 第三步：输入提示词

在海螺AI中输入提示词，例如：

```
请根据以下脚本生成视频：

[粘贴完整的视频脚本内容]

要求：
- 时长：[X] 分钟
- 风格：[科普/故事/种草]
- 画面描述：[已包含在脚本中]
```

### 第四步：生成视频

1. 点击生成按钮
2. 等待AI生成（通常3-10分钟）
3. 预览效果
4. 如需调整，可以修改提示词重新生成

### 第五步：下载视频

1. 点击下载按钮
2. 选择保存位置
3. 建议保存到：`~/Downloads/` 或项目目录

## 视频保存路径

建议保存位置：

```
~/Downloads/
  └── [视频名称].mp4

# 或项目目录
content_automation/
  └── videos/
      └── [视频名称].mp4
```

## 常见问题

### Q: 生成失败怎么办？

1. 检查脚本是否过长（建议分批生成）
2. 修改提示词，更加具体
3. 尝试不同的风格描述
4. 联系海螺AI客服

### Q: 视频太长/太短？

在提示词中明确时长要求：

- 「生成3-5分钟的视频」
- 「控制在2分钟以内」
- 「完整展示，不做删减」

### Q: 画面不符合预期？

细化画面描述：

- 「使用科幻风格」
- 「色调偏冷」
- 「添加动态文字特效」
- 「画面要有冲击力」
